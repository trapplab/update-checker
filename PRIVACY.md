# Privacy Policy
Last updated: 18.02.2026

Update Checker respects your privacy. This Privacy Policy explains how information is handled when you use the app.

## 1. Data Collection

Update Checker does not collect, store, or transmit any personal data.

Specifically:

* No user accounts
* No analytics or tracking
* No advertising SDKs
* No crash reporting
* No backend servers

All app functionality operates entirely on the user's device.

## 2. Internet Usage

Internet access is used to check for app updates and to access web services.

No user data is transmitted to the developer.

## 3. Third-Party Services

Update Checker uses web services from 1000mobiles.info. When using these services, the privacy policy of 1000mobiles.info applies:

* [1000mobiles.info Privacy Policy](https://www.1000mobiles.info/privacypolicy)

Please refer to the linked privacy policy for information on how data is handled when using these web services.

## 4. Children's Privacy

Update Checker does not knowingly collect personal information from children. Since no data is collected, no personal information is stored or shared.

## 5. Open Source Transparency

Update Checker is an open-source application. The source code is publicly available and can be reviewed to verify how the app works and how data is handled.

## 6. Changes to This Policy

This Privacy Policy may be updated from time to time. Any changes will be reflected on this page with an updated revision date.
